주문 제작 · 견적 요청
  • 진열 상품에 원하는 사이즈가 없나요?

    원하는 사이즈와 형태를 알려주세요. 최고의 설계자가 직접 설계해 드립니다.

  • 보유하신 로고를 인쇄하고 싶으신가요?

    인쇄 사양을 보내주시면 인쇄 제작이 가능합니다. (간단한 Logo 배치나 TEXT 제작 가능-무료)

  • 제품에 딱 맞는 포장물 제작을 원하시나요?

    제품 사양과 이미지를 보내주시면 전문가가 직접 포장물 제안해 드립니다.

  • 대량 제작을 원하시나요?

    다년간의 MRO 시장의 경험으로 최적화된 제작 업체를 찾아, 비교 견적을 제공합니다.

  • 포장재의 Total Solution이 필요하신가요?

    패커원은 최고의 역량을 갖춘 전문가와 인프라를 바탕으로 최적의 솔루션을 제안합니다.

  • 별도 SIZE 제작
    및 인쇄추가(Logo)

    · 타입, 골지, 재질, Size, 수량 입력, 디자인 파일 적용
    · 인쇄 추가 : Logo 파일 / 인쇄위치/ 크기 입력
    ※ 박스, 비닐 제작 가능

  • 상품 맞춤 제작

    · 타입, 골지, 재질, Size, 수량 입력
    · 디자인 및 도면 파일 첨부
    ※ 박스만 제작 가능

  • 대량 구매

    · 상품번호, Size, 수량 입력
    ※ 모든 상품 제작 가능
